## Code Highlighting

Nextra supports syntax highlighting for many languages:

```python
def hello_world():
    print("Hello, World!")
    return True
```

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
    return `Hello, ${name}!`;
}
```
