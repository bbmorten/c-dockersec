### Step-by-Step on Ubuntu 24.04


### ğŸ’¡ Optional: Persistent Bridge (via Netplan)

Create `/etc/netplan/01-bridge.yaml`:

```yaml
network:
  version: 2
  renderer: networkd
  bridges:
    br0:
      dhcp4: yes
      interfaces: [eth0]
```

Then apply it:

```bash
sudo netplan apply
```
